ABOUT
ExecEnvHealthCheck v3.2
WHAT IS THIS
ExecEnvHealthCheck is a comprehensive Roblox executor diagnostic suite. It tests security vectors, function coverage, UI library compatibility, and raw performance — generating a self-contained HTML report.
HOW IT WORKS
Run the script inside any executor. It probes 100+ vulnerability vectors, benchmarks 50+ executor functions, stress-tests the Lua VM, and computes big Fibonacci numbers — then writes a detailed .html report.
THIS SITE
Drop your generated .html files here to view and compare them. Reports are stored in your browser's localStorage — fully offline, no server, no uploads. Your data never leaves your machine.
