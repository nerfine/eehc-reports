01 / WHAT
What Is This
ExecEnvHealthCheck is a comprehensive Roblox executor diagnostic suite. It tests security vectors, function coverage, UI library compatibility, and raw performance — generating a self-contained HTML report.
02 / HOW
How It Works
The script runs inside any executor. It probes 100+ vulnerability vectors, benchmarks 50+ executor functions, stress-tests the Lua VM with 6 stages, and computes big Fibonacci numbers — then writes a detailed .html report.
